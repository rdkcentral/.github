## Security Guidelines

### Reporting Security Issues

If you believe you have found a security vulnerability in any RDK Central repository, please report it following our secure disclosure process.

**📋 Complete instructions are available in our Wiki:**
[How to Submit Security Issue](https://wiki.rdkcentral.com/display/RDK/How+to+Submit+Security+Issue)

**🔒 Security Guidelines:**
[RDK Security Guidelines](https://wiki.rdkcentral.com/display/RDK/Security+Guidelines)

### Important Notes

- **DO NOT** create public GitHub issues for security vulnerabilities
- Follow the secure reporting process outlined in our Wiki
- All security reports are reviewed promptly by our security team
- We follow responsible disclosure practices
